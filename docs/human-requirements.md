# 智能终端助手 需求描述

本项目需要使用仓颉（Cangjie）语言构建一个智能体终端助手，调用大模型把用户的自然语言需求转换为可运行的命令，并辅以讲解等，以优化 Linux 用户的终端使用体验。

效果描述：用户在终端里输入 `? 请删除所有std开头的文件夹`，本软件弹出，调用大模型生成出命令 `rm -rf std*`，并输出一段该命令的说明（rm、以及 -r -f 参数的一句话简要说明，以及橙色提示：该命令具有危险性，需要删除文件无法恢复），让用户选择是否运行/给出下一步修改指示。如果运行，程序会 spawn 一个 shell 运行该命令，如果复制，程序将会把命令复制到剪贴板，让用户粘贴并运行。
用户在终端里输入 `? 更新系统`，本软件弹出，调用大模型，大模型提出问题「调查系统使用的包管理器」，请求执行命令 `cat /etc/os-release` 等，用户同意，大模型调查一轮后生成命令 `sudo apt update && sudo apt upgrade`，并储存环境调查结果「用户正在使用 Debian Linux」供之后载入，输出一段该命令的说明（apt 是包管理器；update 更新源；upgrade 更新软件包）以及交互性提示（是否需要让大模型接管终端）和危险提示（命令需要 sudo）（终端助手将完全接管终端），让用户选择是否运行/给出下一步修改指示。如果选择运行和接管，大模型会 spawn 一个 pty 运行该程序，并自动进行交互操作，如果选择仅复制，程序将会把命令复制到剪贴板，让用户粘贴并运行。（普通用户并不可以直接往终端里模拟打字）
